# Cronos API

## Prerequisites

node version >= 20.10.0
npm version >= 10.2.3

## Scripts

### Install all the packages

```bash
npm install
pnpm install
yarn
```

### Setup local environment

```bash
cp .env.example .env
```

### Start development

```bash
npm run dev
```

### Build project

```bash
npm run build
```

### Start and build project

```bash
npm start
```

### Test project

```bash
npm test
```
